# HijriDatePicker


## Installation

> Step 1. Add the jitpack

```javascript
allprojects {
    repositories {
      ...
      maven { url 'https://jitpack.io' }
    }
  }
  ```
  
> Step 2. Add the dependency

  dependencies {
          implementation 'com.github.YasserAdel564:HijriDatePicker:1.0.2'
  }
